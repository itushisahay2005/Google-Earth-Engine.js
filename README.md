# Google Earth Engine Geospatial Portfolios

A collection of Google Earth Engine (GEE) scripts focused on spatial data science, environmental monitoring, and remote sensing analysis.

## 🛠️ Featured Projects

### 1. Regional Vegetation & Desertification Analysis
*   **Objective:** Mapped concurrent regional trends of greening and desertification using long-term satellite imagery.
*   **Data Used:** MODIS/Landsat NDVI time-series raster data.
*   **Key Findings:** Tracked environmental dynamics over two decades to identify significant shifts in vegetation health.

### 2. Urban Hydrological Risk Model
*   **Objective:** Developed a geospatial audit and risk model focusing on urban river corridors.
*   **Data Used:** Digital Elevation Models (DEM), watershed processing datasets, and stream segment tracing.
*   **Key Findings:** Mapped hidden drainage networks and identified areas of urban encroachment vulnerable to flooding.

## 🧰 Tech Stack & Data Sources
*   **Platform:** Google Earth Engine (GEE) API
*   **Languages:** JavaScript / Python
*   **Core Concepts:** Raster processing, Time-series analysis, Terrain analysis, Remote Sensing

## 🚀 How to Run the Code
1. Copy the code from the respective `.js` file in this repository.
2. Paste it into the [Google Earth Engine Code Editor](https://code.earthengine.google.com/).
3. Click **Run** to visualize the interactive maps and charts.
